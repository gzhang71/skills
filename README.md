# Personal Skills

A local Claude Code plugin with productivity skills for interview prep and daily use.

## Install

```bash
claude plugin install /Users/arthur.zhang/code/skills
```

## Skills

### Interview Prep

| Skill | Invoke | Description |
|-------|--------|-------------|
| `interview-repo-review` | `/interview-repo-review` | Orient in an unfamiliar codebase in minutes |
| `interview-feature` | `/interview-feature <description>` | Implement a feature under time pressure |
| `interview-debug` | `/interview-debug <error>` | Systematically diagnose and fix bugs |
| `interview-system-design` | `/interview-system-design <system>` | Structure a system design interview answer |

### Personal

| Skill | Invoke | Description |
|-------|--------|-------------|
| `news-summary` | `/news-summary [topic]` | Daily news briefing across top categories |
| `google-calendar` | `/google-calendar [timeframe]` | Review your upcoming Google Calendar schedule |
